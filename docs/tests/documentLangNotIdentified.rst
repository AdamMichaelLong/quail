===============================
The document must have a "lang" attribute
===============================

*Severity code:* Severe error

.. php:class:: documentLangNotIdentified


The document should have a default langauge, by setting the "lang" attribute in the <code>html element.



Example
-------
Wrong
-----

.. code-block:: html

    &lt;html&gt;



Right
-----

.. code-block:: html

    &lt;html lang="en"&gt;
